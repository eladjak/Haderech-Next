## התקנה 🛠️

```bash
# התקנת תלויות
pnpm install

# הגדרת משתני סביבה
cp .env.example .env.local

# הרצת הפרויקט
pnpm dev
```

## טכנולוגיות 💻

- Next.js 14
- TypeScript
- Supabase
- Tailwind CSS
- OpenAI

## תכונות עיקריות ⭐

- מערכת קורסים אינטראקטיבית

## פיתוח 🔧

```bash
# בדיקות
pnpm test

# בדיקת טיפוסים
pnpm type-check

# בדיקת לינטינג
pnpm lint

# בנייה לייצור
pnpm build
```
