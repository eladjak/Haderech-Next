1

Convex Production

חובה

מה זה Convex?

ה-Backend של האפליקציה — מסד נתונים, פונקציות שרת, real-time sync. צריך לפרוס אותו בנפרד מ-Vercel.

  1. פתחו את דשבורד Convex

    כנסו ל-dashboard.convex.dev ובחרו את הפרויקט colorless-guanaco-894

  2. פרסו לפרודקשן

    מהטרמינל, בתיקיית הפרויקט:

    # Login (if needed) npx convex login # Deploy to production npx convex deploy
  3. הגדירו Environment Variables
    # AI Chat Coach npx convex env set ANTHROPIC_API_KEY sk-ant-xxxxx # Push Notifications npx convex env set VAPID_PUBLIC_KEY BKSeHFW4CVX6sAvImUjsMAaabOOQBCkL-... npx convex env set VAPID_PRIVATE_KEY XOsUbTzKoUbtdHAGV52OLSNeotppIuXQx...
  4. מלאו נתונים ראשוניים (Seed)
    # Insert main course with all 75 lessons npx convex run seedHaderech:seedHaderechCourse
  5. שמרו את ה-Production URL

    העתיקו מ-Settings → Deployment URL. נראה כמו: https://xxx-production.convex.cloud

2

Clerk Production

חובה

הגדרת אימות משתמשים

Clerk מנהל הרשמה, כניסה, ופרופילי משתמשים. צריך Production instance נפרד מ-Development.

  1. פתחו את Clerk Dashboard

    לחצו על "Production" בתפריט העליון (ליד Development)

  2. הגדירו Sign-in Methods

    ב-User & Authentication → Email, Password, Google Sign-in

  3. הוסיפו את הדומיין

    ב-Settings → Domains → הוסיפו haderech.co.il

  4. העתיקו Production Keys

    ב-API Keys → העתיקו:

    pk_live_xxxxxPublishable Key
    sk_live_xxxxxSecret Key
✓
לוקליזציה בעברית כבר מוגדרת בקוד — אין צורך בהגדרה נוספת.
3

Vercel Deploy

חובה

פריסת Frontend

Vercel מריץ את ה-Next.js app. Region מוגדר ל-fra1 (Frankfurt) — הכי קרוב לישראל.

  1. התקינו Vercel CLI
    npm i -g vercel
  2. חברו את הפרויקט
    vercel link

    בחרו את הארגון והפרויקט (או צרו חדש)

  3. הגדירו Environment Variables

    ב-Vercel Dashboard → Settings → Environment Variables, או מהטרמינל:

    # Required vercel env add NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY vercel env add CLERK_SECRET_KEY vercel env add NEXT_PUBLIC_CONVEX_URL vercel env add CONVEX_DEPLOY_KEY vercel env add NEXT_PUBLIC_APP_URL # https://haderech.co.il vercel env add NEXT_PUBLIC_APP_NAME # הדרך vercel env add NEXT_PUBLIC_VAPID_PUBLIC_KEY # Optional vercel env add NEXT_PUBLIC_GA_ID # G-XXXXXXXXXX vercel env add RESEND_API_KEY # re_xxxxx
  4. פרסו!
    vercel --prod
⚠
חשוב: ודאו ש-NEXT_PUBLIC_CONVEX_URL מצביע על ה-Production URL של Convex (לא Development)!
4

DNS & Domain

חובה

חיבור הדומיין

  1. ב-Vercel: הוסיפו דומיין

    Settings → Domains → Add → haderech.co.il

  2. ב-ספק ה-DNS שלכם:
    TypeNameValue
    A@76.76.21.21
    CNAMEwwwcname.vercel-dns.com
  3. המתינו ל-DNS propagation

    בדרך כלל 5-30 דקות. SSL מופעל אוטומטית על ידי Vercel.

5

Stripe תשלומים

אופציונלי

הפעלת תשלומים (כשמוכנים)

התשתית כבר קיימת בקוד. כשתרצו להפעיל תשלומים — הוסיפו את המפתחות.

  1. צרו חשבון Stripe ב-dashboard.stripe.com
  2. צרו Products & Prices

    מסלול בסיסי + מסלול פרימיום → שמרו את ה-Price IDs

  3. הגדירו Webhook

    Endpoint: https://haderech.co.il/api/webhooks/stripe

    Events: checkout.session.completed, customer.subscription.*

  4. הוסיפו Variables ב-Vercel:
    vercel env add STRIPE_SECRET_KEY vercel env add STRIPE_WEBHOOK_SECRET vercel env add NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY vercel env add STRIPE_BASIC_PRICE_ID vercel env add STRIPE_PREMIUM_PRICE_ID
