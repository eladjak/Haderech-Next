# הדרך - פלטפורמת למידה מתקדמת

פלטפורמת למידה חדשנית למציאת זוגיות מאושרת. משלבת סימולטור אינטראקטיבי, פורום קהילתי וקורסים מקוונים. מיועדת לעזור למשתמשים לפתח מיומנויות מקצועיות ואישיות דרך למידה חוויתית ואינטראקטיבית.

## תכונות

- סימולטור תרגול אינטראקטיבי
- פורום קהילתי עם דיונים ושיתוף ידע
- קורסים מקוונים עם מעקב התקדמות
- פרופיל משתמש אישי
- ממשק עברי מלא עם RTL
- עיצוב מודרני ונגיש

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| [Next.js](https://nextjs.org/) | פריימוורק React |
| [TypeScript](https://www.typescriptlang.org/) | בטיחות טיפוסים |
| [Tailwind CSS](https://tailwindcss.com/) | עיצוב |
| [Framer Motion](https://www.framer.com/motion/) | אנימציות |

## התקנה

```bash
git clone https://github.com/eladjak/Haderech-Next.git
cd Haderech-Next
bun install
bun dev
```

פתחו [http://localhost:3000](http://localhost:3000) בדפדפן.
